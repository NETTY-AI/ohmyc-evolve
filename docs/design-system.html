14 — TEMPERATURE

Bond Warmth

관계의 온기를 체온에서 시작하는 은유로 표현. 스트릭 대신 — 죄책감이 아닌 그리움.

Close Friend

36.5°C

COOL
34.0°C
Been Away
자연스럽게 식어감 · 기억은 보존
TEMPERATURE vs STREAK
Temp — 점진적 하강 · "식었다" · 그리움
Streak — 이진 끊김 · "끊겼다!" · 죄책감
Temp — 기억이 있어 빨리 회복
Streak — 0부터 다시
COLOR GRADIENT
Cool (34°)Baseline (36.5°)Warm (39°+)
높은 온도 = 활발한 발현 → 더 풍부하게, 더 자기답게
낮은 온도 = 축소된 발현 → 조심스럽게, 다시 알아가듯

18 — ACCESSIBILITY

Inclusive Design

모든 사람이 캐릭터와 관계를 맺을 수 있어야 한다.
REQUIREMENTS
✓ 색상 대비 WCAG AA (4.5:1)
✓ 터치 타겟 최소 44x44pt
✓ Dynamic Type 지원
✓ VoiceOver 모든 요소 label
✓ 최소 텍스트 11px
